== Description ==

MDC Target Blank forces links in posts or pages to be opened in a new tab. You don't have to write target="_blank" manually! Cheers..

== Installation ==

1. Upload the `mdc-target-blank` folder and its contents to your `wp-contents/plugins` directory.
2. Activate in the `Plugins` menu.
4. You are done!
